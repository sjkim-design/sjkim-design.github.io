#### 4. 벡터

벡터는 생성과 사용 두 가지 측면에서 볼 수 있음

- **생성**

  벡터는 한 가지 타입만 저장 가능

  

{% highlight r %}

v <- c(1,2,'3')

{% endhighlight %}  

```
"1" "2" "3"
```



{% highlight r %}

v <- c(1:5)

{% endhighlight %}  

```
[1] 1 2 3 4 5
```



- **사용**

[1] 선택

벡터안에는 위치 정보나 벡터이름을 적어 줘야함 
v[1] # 원하는 위치의 값을 뽑아옴
v[c(1,2)]
v[-3] # 3번째꺼 제외하고 뽑기



[2] 벡터에서 많이 사용하는 함수 paste, sep, repeat


 paste('a','b',sep ="-")

```
"a-b"
```
